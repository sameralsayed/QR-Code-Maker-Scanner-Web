# QR Code Maker & Scanner Web

## Description
Exact sleek web replica of the popular Google Play app "QR Code Maker & Scanner" by Tool Apps Hub (1M+ downloads).  
Fully functional in-browser QR & barcode scanner + generator with camera, gallery, history, flashlight simulation, color customization and instant downloads. Everything runs privately in your browser – no data leaves your device.

## Features
📷 Instantly scan QR codes & barcodes with camera  
🖼️ Scan from Gallery (upload any photo)  
📜 Scanning History Saved (persistent in browser)  
🔦 Flashlight & Auto Zoom support  
✨ Create custom QR codes for URLs, WiFi, vCard, Contacts  
🎨 Fully customizable colors & shapes  
📥 Export as PNG (PDF ready)  
🔒 Safe & Secure – 100% private offline-capable web app
